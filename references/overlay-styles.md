# Text Overlay Style Presets

All styles are generated by `scripts/generate_titles.py`. Pass `--style <name>`.

## dark-pill (default)

The standard look — dark translucent pill, white text, rounded corners.
Works on both light and dark content.

```python
SCRIM    = (10, 12, 20, 165)   # near-black, ~65% opacity
TEXT     = (255, 255, 255, 255) # white
RADIUS   = 26
PAD_X    = 44
PAD_Y    = 30
FONT_SIZE = 52
```

Canvas: 1080px wide, dynamic height (text_height + 2×PAD_Y + 8px).
Pill is centered horizontally; 4px top gap before the pill starts.

## light-pill

Same shape, inverted colours. Use on dark content (terminal, code editor, night-mode UIs).

```python
SCRIM    = (255, 255, 255, 200) # white, ~78% opacity
TEXT     = (10, 12, 20, 255)   # near-black
RADIUS   = 26
PAD_X    = 44
PAD_Y    = 30
FONT_SIZE = 52
```

## bottom-bar

Full-width bar across the bottom of the frame. Good for longer text or subtitle-style overlays.

```python
SCRIM    = (10, 12, 20, 200)   # near-black
TEXT     = (255, 255, 255, 255)
RADIUS   = 0                   # no rounding — full edge-to-edge bar
PAD_X    = 60
PAD_Y    = 36
FONT_SIZE = 48
CANVAS_W = 1080                # full width, transparent left/right edges not needed
```

Overlay position: `y = OUTPUT_H - canvas_height - 40` (40px above the bottom edge).

## accent-pill

Same as dark-pill but with a brand colour scrim. Pass `--accent-color "R,G,B"`.

```python
SCRIM    = (R, G, B, 180)     # user-specified RGB, ~70% opacity
TEXT     = (255, 255, 255, 255)
RADIUS   = 26
PAD_X    = 44
PAD_Y    = 30
FONT_SIZE = 52
```

## Overlay position in the video

Default y position: **y=140** — just below the top UI chrome (browser tab bar, app title bar).
This is safe across most screen recording content.

For bottom-bar style, calculate dynamically:
```
y = output_height - title_canvas_height - 40
```

## Two-line rule

Always split overlay text into 2 lines. Keep each line under ~40 characters.
Long single-line text causes the script to shrink the font — splitting reads better.

Good:
```
"A hosted payment page\nbuilt for AI agents"
"HTTP 402 → signed\nsettled on Base mainnet"
```

Bad (too long for a single line):
```
"The hosted payment page is built specifically for AI agents using x402"
```
